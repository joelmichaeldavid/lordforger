# LordForger Documentation Map

This folder is the canonical home for project planning and design notes.

## Start Here
- `LordForger_Project_Sources.md` — consolidated foundational notes from studio charter through systems architecture.

## Suggested Growth Structure
- `design/` — system design docs (hex, encounters, time, settlement, economy, progression)
- `narrative/` — lore, factions, regions, character arcs
- `production/` — roadmap, milestones, issue maps, sprint planning

## Rule
When a decision is made in chat, copy the durable part into docs so future agents can use the same source of truth.
