# Session 1 — Vertical Slice Kickoff

## Goal
Deliver a playable micro-loop immediately: **travel → encounter → world reaction**.

## What was built
- A browser-playable prototype in `prototype/` with:
  - hex travel
  - encounter outcomes
  - time advancement (segments + day rollover)
  - Hearthvale safety/prosperity reactions
  - rumor snippets driven by encounter outcomes

## How to run
Open `prototype/index.html` in a browser.

## Next tasks (department lanes)

2. **Narrative**: rumor templates per faction and biome.
3. **Engineering**: extract game state into modular files and add save/load.
4. **UX**: improve hex readability and consequence messaging.
5. **Production**: convert tasks into GitHub issues with acceptance criteria.
