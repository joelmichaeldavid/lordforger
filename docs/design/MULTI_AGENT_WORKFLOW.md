# Multi-Agent Workflow (LordForger)

## Lanes
- Systems Design
- Narrative
- Engineering
- UX
- QA
- Production

## Working agreement
- Each lane works from a dedicated issue and branch.
- Small PRs; merge often.
- Shared truth lives in `docs/`.

## Definition of done (prototype feature)
- Playable in browser
- Logged world consequences
- Notes updated in docs
- Follow-up tasks created
